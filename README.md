## Features
- Registration and authorization form via JWT token
- Add / Rename / Remove channels
- Send and receive messages in real time
- Switching between channels

## Install locally

```bash
$ make install
```

## Run App locally

```bash
$ make local start
```                     

## Deploy

```bash
$ make build
```
